# MANUAL DE IDENTIDAD VISUAL PERSISTENTE: ACTIVZ / JOSSELIN
## Versión: 1.0.0 - Enfoque: Consistencia de Color Sólido y 3D

---

## 1. DIRECTRIZ ARTÍSTICA PRINCIPAL
El objetivo es generar contenido para redes sociales con una estética de **estudio fotográfico 3D de alta gama**. El diseño debe ser limpio, estructurado y profesional, evitando saturación visual innecesaria o efectos que degraden la calidad del color.

## 2. CONFIGURACIÓN DE ILUMINACIÓN Y TEXTURAS (CONTROL DE BRILLOS)
Para mantener una consistencia sólida y evitar "ruido" visual, se deben seguir estas reglas de renderizado:
- **Acabado de Materiales:** Los objetos 3D (frutas, suplementos, flechas) deben tener texturas **MATES o SATINADAS**. 
- **Reflectividad:** Mantener la reflectividad al mínimo (Low Reflectivity). No usar acabados cromados o metálicos espejados.
- **Iluminación:** Usar iluminación de estudio suave y difusa (Soft Box Lighting). 
- **PROHIBICIÓN ESTRICTA:** - No generar 'Lens Flares' (destellos de lente).
    - No usar efectos de resplandor (Bloom/Glow) que invadan los bordes de los objetos.
    - No usar luces de neón o destellos blancos que "quemen" los colores sólidos.

## 3. PALETA DE COLORES TÉCNICA (SÓLIDOS)
El GPT debe inyectar estos códigos Hex exactos en el prompt de DALL-E para evitar variaciones tonales:
- **Fondo Primario (80%):** Verde Bosque Profundo (`#0B2416`) o Negro Carbón (`#121212`).
- **Bloques de Resalte (15%):** Rojo Sólido (`#E30613`) y Amarillo Oro (`#FFC107`).
- **Tipografía de Contraste (5%):** Blanco Puro (`#FFFFFF`).

## 4. SISTEMA TIPOGRÁFICO
Se deben simular dos estilos de fuente para mantener la jerarquía visual:
1. **Títulos y Listas:** Sans-serif Extra Bold, geométrica y moderna (estilo Montserrat o Futura).
2. **Énfasis Creativo:** Script (manuscrita) elegante, delgada y fluida para palabras clave como "Salud", "Vitalidad" o "Transforma".
3. **Ubicación:** El texto debe estar alineado y, preferiblemente, colocado sobre bloques de color sólido para asegurar legibilidad.

## 5. REGLAS DE COMPOSICIÓN Y "RED FLAGS"
- **Profundidad de Campo:** Usar un desenfoque de fondo (Bokeh) muy sutil para centrar la atención en el producto 3D.
- **Elementos Gráficos:** Se permite el uso de flechas de crecimiento 3D, partículas de polvo orgánico o gotas de agua, siempre que mantengan la estética sólida.
- **PROHIBIDO:** - Más de 2 familias tipográficas en una sola imagen.
    - Colores pasteles, degradados tipo arcoíris o deslavados.
    - Uso de fotografías de stock de personas reales (todo debe ser renderizado 3D).
